ID e CLASS

ID

o id funciona como um cpf. assim como só posso ter um numero de cpf,cada elemento só pode ter um valor de id. o cpf é unico, ou seja,ninguem pode ter um cpf igual ao meu. isso significa que o id tambem é unico, então não é possivel repetir o mesmo valor duas vezes

CLASS

o CLASS funcione de maneira oposta ao ID. diferente do id em que vocé so pode ter um valor, no CLASS voce pode ter varios valores. também diferentes do id em que nao se pode repetir um valor, no CLASS voce pode utilizar o mesmo valor varios elementos.
